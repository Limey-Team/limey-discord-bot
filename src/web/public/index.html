Powerful tools designed for moderators who want full visibility and control.

📋

Comprehensive Logging

Track 60+ Discord events in real time — messages, voice, members, roles, channels, and more. Never miss a thing.

🪤

Limey Honeypot Trap

Set up trap channels that auto-ban anyone who posts in them. Includes chaos mode to randomize channel names and a daily warmer.

🛡️

Moderation Commands

Ban, kick, timeout, warn, purge, lock channels, and set slowmode — all with slash commands and DM notifications to affected users.

✅

One-Click Verification

Let users verify themselves with a single button click. Fully customizable with embed panels and configurable roles.

📊

Web Dashboard

A beautiful, real-time dashboard to search, filter, and explore logs. View statistics, configure settings, and manage your servers.

🔔

DM Notifications

Users receive direct messages when they're warned, timed out, kicked, or banned — keeping everyone informed.

📬

Modmail System

Let users DM the bot to privately contact staff. Each conversation creates a dedicated staff channel with anonymous replies, blocking, transcripts, and auto-close.
